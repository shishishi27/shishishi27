hexo+obsidian+github搭建个人blog

Hexo安装

需先安装Node.js和Git。

新建blog文件夹，右键选择Open Git Bash Here，使用npm安装Hexo。

查看是否安装成功：

正常会输出版本号及详细信息，若出现“hexo: command not found”

“$PATH:/c/Users/用户名/AppData/Roaming/npm”可在“npm list -g hexo-cli”的输出中查看。

访问“http://localhost:4000”，出现 Hexo 默认页面，表示本地博客安装成功！

GitHub Pages创建

GitHub新建仓库，Repository name 中输入 “github用户名.github.io”。设置为”Public”及勾选”Initialize this repository with a README”。

进入该仓库，复制http链接。

部署 Hexo 到 GitHub Pages

首先安装hexo-deployer-git：

打开_config.yaml文件,，修改文件的URL和Deployment部分：

完成后运行将网站上传部署到 GitHub Pages。

进入Settings -> Pages，选择deploy的branch为master，刷新网页，获取个人主页网址https://github用户名.github.io 。

完成！这时访问我们的 GitHub 域名 https://github用户名.github.io 就可以看到 Hexo 网站了。

工作流

1.新建blog

2.obsidian打开文件夹开始写作

3.终端本地预览与发布
